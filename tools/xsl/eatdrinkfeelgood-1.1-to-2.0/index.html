eatdrinkfeelgood 1.1 to 2.0

This stylesheet converts a document conforming to the eatdrinkfeelgood 1.1 DTD into a document conforming to the experimental eatdrinkfeelgood 2.0a format.

The current version of the eatdrinkfeelgood 1.1 to 2.0 stylesheet is 0.1.

released April 01, 2006

source

www.eatdrinkfeelgood/src/xsl/eatdrinkfeelgood-1.1-to-2.0a-v0.1.tar.gz

example

xsltproc -o /path/to/recipe.n3 \
 /path/to/eatdrinkfeelgood-1.1-to-2.0a.xsl \
 /path/to/recipe.xml
