these are eatdrinkfeelgoodperl tools

edfgml-perl-tools

The current version of the edfgml-perl-tools package is 0.1.1.

Released January 25, 2002

These tools are specific to the Eatdrinkfeelgood 1.0 DTD and are no longer being actively maintained.

source

www.eatdrinkfeelgood.info/src/perl/edfgml-perl-tools-0.1.1.tar.gz

edfgml-perl-tools/bin

  • make-index-card

    • Create an index-card size PDF document from one or more paths containing Eatdrinkfeelgood documents.

edfgml-perl-tools/cgi

  • pdf-gen

    • A CGI script to create an index-card size PDF document from one or more paths containing Eatdrinkfeelgood documents.

edfgml-perl-tools/lib

  • Eatdrinkfeelgood/Apache.pm

    A mod_perl handler for displaying, and generating directory indices of Eatdrinkfeelgood documents using XSLT.

  • Eatdrinkfeelgood/Directory/Index.pm

    I would have liked to use Petr Cimprich's XML::Directory::SAX and I may yet. I am sticking with this module for the time being because it has better support for including/excluding files and because there didn't seem to be any (simple) way to get XML::Directory::SAX to stream files alphabetically.

    note: most of these problems have been fixed in version 0.96 of XML::Directory

  • Eatdrinkfeelgood/PDF/Base.pm

    Wrapper module for PDFLib.pm

  • Eatdrinkfeelgood/PDF/IndexCard.pm

    Object methods for generating index card sized PDF files from one or more Eatdrinkfeelgood documents.

  • Eatdrinkfeelgood/Recipe.pm

    Object methods for reading an Eatdrinkfeelgood recipe.

  • Eatdrinkfeelgood/XML/YAGenerator.pm

    This module subclasses XML::Generator to fix some bugs in the creation of attributes and processing instructions.
